# PC7 – Reduced Basis Methods for Parametric PDEs

This collection of four notebooks constitutes the seventh practical assignment of *Machine Learning for Scientific Computing and Numerical Analysis*. It implements the Reduced Basis (RB) Galerkin method across various parametric PDEs.

## Topics Covered

1. **PC7_01_Finale.ipynb** – Poisson equation
   - Building the reduced basis via SVD
   - Galerkin projection onto the RB space
   - Error estimation and convergence study

2. **PC7_02_Finale.ipynb** – Elliptic PDE with varying coefficients
   - Parameter-dependent stiffness assembly
   - Offline–online decomposition
   - Validation across parameter samples

3. **PC7_03_Finale.ipynb** – Heat equation
   - Time-dependent RB space construction
   - Snapshot generation for space–time manifold
   - Online evaluation and error tracking

4. **PC7_04_Finale.ipynb** – Viscous Burgers equation
   - Nonlinear PDE treatment via RB
   - Handling affine and non-affine parameter dependence
   - Case studies and accuracy benchmarks
